 Functions

(I) Automodding capabilities include a word filter and custom responses based on @mentions. The configuration is entirely granular and can be configured with methods explained in the help commands.

(II) Direct messasge capabilities allow a message to be sent from a channel to a user and for the responses the bot receives to be sent to that channel.
